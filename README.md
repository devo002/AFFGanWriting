## Installation

```console
conda create --name AFFWriting python=3.7

conda install pytorch==1.4.0 torchvision==0.5.0 cudatoolkit=10.1 -c pytorch
git clone https://github.com/aimagelab/VATr.git && cd VATr
pip install -r requirements.txt
